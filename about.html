Sobre Mim

Sou um profissional de Análise de Dados com mais de cinco anos de experiência em transformar dados brutos em decisões de negócio estratégicas. Meu toolkit técnico abrangente inclui SQL Server/DBeaver, Python, Power BI, Git e GitHub, que utilizo para criar soluções robustas.

Atualmente, minha função principal é atuar como a ponte estratégica entre a equipe de liderança e a inteligência de dados. Sou responsável pela gestão da plataforma de concentração dos OKRs (Objectives and Key Results), implementando rotinas de atualização e integração através de APIs e scripts em Python para fornecer dados e insights acionáveis que orientam o planejamento estratégico. Minha experiência é diversificada, incluindo um período significativo no setor industrial. Nessa área, analisei dados de telemetria de máquinas para otimizar serviços de manutenção e reduzir custos operacionais. Esse trabalho gerou valor direto ao garantir maior disponibilidade de produção e forneceu insights precisos tanto para os setores de Manutenção quanto de Produção.

Além disso, possuo experiência na construção de soluções de coleta e processamento de dados, desde o uso de ferramentas Low-Code (como Power Apps) para captura de dados em campo e geração de análises detalhadas, até a automação de processos. Desenvolvi RPAs (Robotic Process Automation) para realizar a captura de dados do sistema SAP ECC, executando a ingestão, transformação e processamento dos dados, garantindo assim uma base limpa e pronta para a geração de insights complexos.
