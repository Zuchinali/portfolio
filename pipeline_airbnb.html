Airbnb Data Pipeline & Investment Strategy: Otimização de ROI via Dados

Ferramentas:
Python, Pandas & NumPy, Streamlit, Parquet, Plotly & Folium, Git & Github e ETL Architecture
Função:
Estruturar um pipeline de dados
Duração:
10 horas

Visão Geral

Este projeto simula uma consultoria de investimentos imobiliários "Data-Driven". O objetivo não foi apenas visualizar dados, mas criar um produto analítico que responde à pergunta: "Onde investir capital para obter o maior retorno com o menor risco em aluguéis de curta temporada?". Através de dados públicos do Inside Airbnb, construí um pipeline que ingere, sanitiza e enriquece os dados brutos, culminando em uma ferramenta de recomendação prescritiva para investidores.

Arquitetura e Metodologiaa

O projeto foi estruturado seguindo boas práticas de Engenharia de Software e Dados, segregando responsabilidades em módulos:

  1. Ingestão e Processamento (ETL)
    • Pipeline de Limpeza Estatística: Implementação de regras de IQR (Intervalo Interquartil) para remoção automática de outliers de preço, garantindo que imóveis de luxo extremo ou erros de cadastro não enviesem a análise de mercado.
    • Engenharia de Atributos (Feature Engineering):
      • Cálculo de Receita Estimada: Desenvolvimento de métricas sintéticas (Ocupação x Diária) baseadas no "San Francisco Model" para estimar o ROI mensal de cada propriedade.
      • Valoração de Amenities: Algoritmo que processa listas de amenities não-estruturadas para isolar o impacto financeiro de itens específicos. Resultado: Identificou-se que o Ar Condicionado agrega +28.9% ao valor da diária, superando a Piscina (+16.0%).
  2. Armazenamento Otimizado
    • Uso de Parquet com compressão Snappy em vez de CSV para a camada processed, garantindo tipagem forte dos dados (schema enforcement) e leitura de alta performance para o frontend.
  3. Visualização e Entrega (Data App)
    • Desenvolvimento de uma aplicação web com Streamlit, permitindo que o usuário final filtre oportunidades por orçamento e visualize "Zonas de Calor de Rentabilidade" em mapas interativos.
    • Matriz de Oportunidade: Um gráfico de dispersão estratégico que isola bairros no quadrante "Baixo Custo / Alto Retorno".
    • Vale ressaltar que também seria possível utilizar o Power BI ou Looker ao invés do Streamlit.

Resultados de Negócio (Consultoria)

A análise automatizada identificou o micro-mercado Waterfront Communities - The Island como alvo ideal de investimento, apresentando uma eficiência de custo superior à média da cidade com uma quebra de receita mínima em relação ao mercado de luxo.

Roadmap de Engenharia (Próximos Passos)

Este projeto é um organismo vivo. As próximas etapas focam na transição da Análise para a Engenharia de Dados Robusta:

  • [Pendente] Containerização: Dockerizar a aplicação e o pipeline ETL para garantir reprodutibilidade e portabilidade entre ambientes (Dev/Prod).
  • [Pendente] Orquestração: Implementação de Docker Compose para gerenciar serviços e futuros bancos de dados.
  • [Pendente] CI/CD: Configuração de GitHub Actions para automação de testes e build de imagens Docker a cada push.
